# 笔记

## c语言数据类型

`int double float char bool`

## 输入输出格式化控制符

* %f ---- float
* %d ---- int
* %c ---- char

## 数据的表现形式

* 常量
  * 整型常量    100 0  -12
  * 实型常量    123.21  123e3
  * 字符常量    'a' \' \"" \\ \? \f \n
  * 字符串常量  "123"
  * 符号常量    #define PI 3.14
* 变量
  * 常变量  const float pi = 3.14;

## 标识符

    组成：字母-数字-下划线，第一个字母为字母或下划线
